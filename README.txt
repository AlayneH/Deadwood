# Deadwood
<hr>
> Deadwood is a virtual board game where players aim to reach stardom. Players travel around the Deadwood backlot in search of acting jobs that earn them money and credit. Players must manage their time and resources to become the best actors on the backlot!

>This implementation of Deadwood was created in Western Washington University's Object Oriented Design Class. It is a virtual version of a board game of the same name by Cheapass Games. The free edition of the original board game can be found [here](https://cheapass.com/free-games/deadwood/)

# Running the Game
<hr>
> Compile deadwood.java and follow the on screen instructions to play
